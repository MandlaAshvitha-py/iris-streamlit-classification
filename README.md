# 🌸 Iris Flower Classification using Streamlit

A Machine Learning web application that predicts the species of an Iris flower based on its measurements using a trained Random Forest Classifier. The application is built with Streamlit and provides an interactive interface for exploring the dataset, visualizing data, training the model, and making predictions.

---

## 📌 Project Overview

This project demonstrates a complete Machine Learning workflow:

- Data Exploration
- Data Visualization
- Model Training
- Model Evaluation
- Flower Species Prediction

The application predicts one of the following Iris species:

- 🌸 Setosa
- 🌸 Versicolor
- 🌸 Virginica

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/MandlaAshvitha-py/iris-streamlit-classification
```

### 2. Navigate to the Project Folder

```bash
cd iris-streamlit-classification
```

### 3. Install Required Packages

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
streamlit run app.py
```

## 📊 Dataset

The Iris dataset contains:

- Total Samples: **150**
- Features: **4**
- Classes: **3**

### Input Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target Classes

- Setosa
- Versicolor
- Virginica
